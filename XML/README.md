# BD_LAB2

### Author: Maria Shyn




Мета лабораторної роботи: здобуття практичних навичок створення програм, орієнтованих на обробку XML-документів засобами мови Python.



Завдання роботи:
  
- Виконати збір інформації зі сторінок Web-сайту за варіантом
  
- Виконати аналіз сторінок Web-сайту для подальшої обробки текстової та графічної інформації, розміщеної на ньому.
  
- Реалізувати функціональні можливості згідно вимог.



Інструментарій:
  
- Мова програмування – Python 2.7
  
- Бібліотека для обробки XML – lxml.
  
- Середовище розробки програмного забезпечення – PyCharm Community
